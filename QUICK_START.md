# 🚀 快速开始：直接重命名原始文件

## ⚠️ 重要：正确的使用流程

### 🎯 必须按照以下步骤操作

```
步骤 1: 选择工作目录 (点击 "Select Directory" 按钮)
   ↓
步骤 2: 从该目录拖拽文件
   ↓
步骤 3: Preview Rename
   ↓
步骤 4: Execute Rename (Direct) ✨
   ↓
✅ 完成！原文件已被重命名
```

---

## 📝 详细步骤

### Step 1: 启动应用（使用 Chrome 或 Edge）

```bash
npm start
```

**浏览器要求：**
- ✅ Chrome 86+
- ✅ Edge 86+
- ❌ Safari（不支持）

访问：http://localhost:5173

### Step 2: 选择工作目录 ⭐ **最重要的一步**

**您会看到蓝色提示框：**
```
🎯 Start Here: Select Your Working Directory
```

**操作：**
1. 点击 **"📁 Select Directory"** 按钮
2. 选择您的文件所在目录
3. 授予读写权限

**成功后显示：**
- `📁 Working in: Photos`
- `✨ Direct File Mode Enabled!`

### Step 3: 从该目录拖拽文件 ⚠️

**重要：** 必须从选择的目录拖拽！

### Step 4: Preview & Execute

1. Preview Rename
2. 点击 **"✨ Execute Rename (Direct)"**
3. ✅ 完成！

---

**记住：必须先选择目录！**
